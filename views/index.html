ISQA_5 - Nasdaq Stock Price Checker

Example usage:

In the route: /api/stock-prices you must enter a query parameter called stock with the name of a stock or you can also enter two stock names separated by &. You can also enter a parameter called like with true value (Only one like is accepted to give by IP). If the stock exists, the server returns an object with the field stockData and as a value it returns another object with the fields stock, price and likes. If several stocks were entered in the URL, the server returns an object with the field stockData and as a value it returns an array of objects with fields stock, price and rel_likes. The value of the rel_likes field corresponds to the difference between the number of likes of both stocks.

/api/stock-prices?stock=GOOG

/api/stock-prices?stock=GOOG&like=true

/api/stock-prices?stock=GOOG&stock=MSFT

/api/stock-prices?stock=GOOG&stock=MSFT&like=true

Example return:

{"stockData":{"stock":"GOOG","price":786.90,"likes":1}}

{"stockData":[{"stock":"MSFT","price":62.30,"rel_likes":-1},{"stock":"GOOG","price":786.90,"rel_likes":1}]}
